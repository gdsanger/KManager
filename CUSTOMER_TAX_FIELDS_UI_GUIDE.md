## Badge Color Scheme

- **Country Code**: Gray (`bg-secondary`)
- **EU Badge**: Blue/Info (`bg-info`)
- **Business Badge**: Blue/Primary (`bg-primary`)
- **Private Badge**: Gray (`bg-secondary`)

## Field Descriptions

| Field | Type | Default | Required | Description |
|-------|------|---------|----------|-------------|
| vat_id | CharField(32) | - | No | VAT ID, auto-uppercase |
| country_code | CharField(2) | 'DE' | Yes | ISO Alpha-2 code |
| is_eu | Boolean | False | Yes | EU customer flag |
| is_business | Boolean | True | Yes | Business/Private |
| debitor_number | CharField(32) | - | No | Accounting number |
