SalesDocument Detail View

PDF voransehen
New Feature: PDF Preview Button
  • Location: SalesDocument Detail View page actions (top right)
  • Button Label: "PDF voransehen" (PDF Preview)
  • Icon: PDF file icon
  • Behavior: Opens PDF preview in a new browser tab (target="_blank")
  • Endpoint: /auftragsverwaltung/documents/{id}/preview/
  • Read-only: No status changes, no number assignment, no snapshots
  • Filename: {document_type.name}_{id}.pdf (e.g., "Rechnung_1.pdf")
Implementation Complete

The PDF preview button has been successfully added to the SalesDocument detail view. It appears between the "Speichern" (Save) and "Abbrechen" (Cancel) buttons in the page actions area.
